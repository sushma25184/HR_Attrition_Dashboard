# HR Attrition Dashboard (Power BI)

## Project Overview
This project is an **HR Attrition Analytics Dashboard** created using **Microsoft Power BI**.  
It helps understand employee attrition by showing key KPIs and department-wise insights in a single view.

## Objectives
- Track overall employee attrition metrics
- Identify department-wise attrition patterns
- Enable quick filtering and analysis for HR decision-making

## Key KPIs (DAX Measures)
- **Total Employees**
- **Employees Left**
- **Employees Stayed**
- **Attrition Rate (%)**

## Dashboard Features
- KPI Cards for quick summary
- Department-wise chart (Attrition Yes/No comparison)
- Department slicer for interactive filtering

## Tools & Technologies
- **Power BI Desktop**
- **DAX**
- (Optional) **Power Query** for basic cleaning

## Dataset
HR Attrition dataset (CSV) with fields like:
Department, Attrition (Yes/No), Employee information, etc.

## How to Use
1. Download the `.pbix` file from this repository.
2. Open it using **Power BI Desktop**.
3. Use slicers/filters to explore department-wise insights.

## Output
- Power BI project file: `.pbix`
